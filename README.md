# Screenshot Tool

> Capture full-screen, delayed, or custom-region screenshots and save as PNG files.

---

## Features
- Full-screen screenshot
- Delayed screenshot (set seconds)
- Region/crop screenshot (x, y, w, h)
- Auto-named output files with timestamp

## Requirements
```
pip install pyautogui
```

## Installation
```bash
git clone https://github.com/brainiacweb-tech/screenshot-tool
cd screenshot-tool
pip install -r requirements.txt
```

## Usage
```bash
python main.py
```
